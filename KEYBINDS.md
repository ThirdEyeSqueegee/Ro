# Default Keybinds

- `H`: Summon horse
- `L`: Toggle lantern
- `V`: Dual wield parry (`Back/Wait` button on controller)
- `X`: Toggle compass
- `Y`: Remote Interaction
- `F3`: Where Are You? menu
- `F5`: Quicksave
- `F7`: ENB FPS counter
- `F8`: QUI
- `F10`: Disable ENB
- `F11`: ENB menu
- `Backspace`: Immersive Equipment Displays menu
